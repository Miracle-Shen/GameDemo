## 1. 系统定位

### 1.1 挖矿小游戏在游戏中的角色

无尽矿井是矿场增长游戏中的主动操作玩法，是主矿场挂机增长之外的额外收益入口。

玩家从矿区主页进入无尽矿井，消耗 `1` 点体力开启 `1` 局挑战。进入后，系统固定发放 `20` 个木镐子作为本局基础探索资源，玩家通过点击砖块和使用范围工具向下推进，在有限基础资源条件下争取更深层数和更高收益。

核心链路：

```text
矿区主页 → 点击无尽矿井入口 → 消耗1点体力入场
    ↓
系统发放本局固定20个木镐子
    ↓
点击砖块 / 使用炸弹 / 获取掉落
    ↓
继续向下推进并提升深度阶段
    ↓
工具耗尽
    ↓
主动退出或工具耗尽 → 本局结算 → 收益回流主矿场
```

### 1.2 设计目标

| 目标 | 说明 |
| --- | --- |
| 模式身份清晰 | 这是主矿场内的扩展模式，不独立产生新经济体系 |
| 开局成本简单 | 1 点体力进入 1 局，单局固定 20 个木镐子，规则直接易懂 |
| 深挖收益明确 | 越深层收益期望越高，让玩家有持续推进动力 |
| 决策张力成立 | 玩家需要在继续下挖和及时止盈之间做判断 |
| 资源模型统一 | 所有收益统一进入主矿场背包、钱包和材料系统 |

### 1.3 与矿区系统的关系

```text
矿区主页 → 点击「无尽矿井」→ 无尽矿井玩法（本页面）
         ↘ 点击其他建筑/系统 → 原有矿区玩法
```

无尽矿井沿用主矿区已确定的系统口径：

- 顶部继续展示体力、主货币、付费货币等主系统信息
- 消耗体力规则与主矿区一致
- 收益统一进入主背包和主货币体系
- 不单独创建模式专属售卖价格体系
- 矿品价值统一参考既有矿品表

## 2. 玩法总览

### 2.1 一句话玩法定义

无尽矿井是一种“消耗 1 点体力进入、每局固定 20 个木镐子开局、通过深度推进争取更高收益”的下挖式挖矿玩法。

### 2.2 单局与深度推进的关系

- `1` 点体力 = `1` 局无尽矿井
- `1` 局无尽矿井 = 以固定 `20` 个木镐子开局的一次连续下挖挑战
- 深度推进 = 本局中持续向更深区域探索并提升收益阶段

也就是说，体力限制开局次数，镐子定义本局基础探索次数，深度决定本局收益上限。

布局更新说明：

- 页面视觉主体必须是中部 `7×6` 挖掘区域，而不是顶部信息卡。
- 深度信息不再以顶部独立大卡片展示，而是直接贴近棋盘中的深度线展示。
- `补给积分` 需要从顶部信息区下移到挖掘区域正上方，作为轻量附属状态条，不可抢占棋盘主视觉。
- 工具操作区优先采用右侧纵向独立工具条，保留移动端单手点击路径。

### 3.2 顶部 HUD

顶部 HUD 延续矿区主页的设计，不单独创造新口径。

| 元素 | 说明 |
| --- | --- |
| 返回按钮 | 左上返回入口，点击后弹出确认框 |
| 体力 | 展示当前剩余体力，进入本模式时消耗 |
| 金币 | 主系统通用货币 |
| 玉币 | 主系统付费货币 |
| 设置 | 打开设置弹层 |

### 3.3 深度与目标信息区

该区域是本玩法的信息锚点，玩家进入后应立即看到当前推进状态。

| 元素 | 说明 |
| --- | --- |
| 当前深度类型 | 展示当前所处深度类型，如地表矿区、深地表、浅海、深海、地心火山泥 |
| 本局推进 | 展示当前深度类型下的补给积分进度，如 `4/12` |
| 本局状态 | 展示如 `推进中`、`工具耗尽`、`结算中` 等短状态文案 |
| 本局收获展示 | 展示本局累计获得的金币、矿石、材料等资源，仅供查看，不承载点击操作 |

布局约束：

- 顶部信息区用于承载“状态信息 + 本局收获累计展示”，不再承载大尺寸深度数值卡片。
- 顶部区域不放置高频点击道具按钮。
- 玩家在挖掘过程中应能持续看到本局收益增长，但不需要在顶部进行任何工具操作。
- 宝箱不在顶部单独累计展示；宝箱开启后，开出的矿石、材料、金币或工具直接并入对应展示位。
- `当前深度类型 / 本局推进 / 本局状态` 在移动端必须保持同一行展示，`收队返场` 允许独立占下一行。
- `金币 / 矿石 / 材料` 在移动端必须保持同一行展示，不允许在窄屏下自动拆成纵向三行。
- 顶部信息区中的数值卡、收获卡均应收紧高度，避免压缩棋盘可视面积。

### 3.4 中部矿井主场景

中部不是纯棋盘，而是“矿井环境 + 可交互砖块区域”的组合。

场景表现应包含：

- 岩层和矿层变化
- 木制支撑结构
- 灯光与尘土粒子
- 不同深度类型的环境色、背景元素和氛围变化
- 下挖推进的视觉位移感
- 采矿区域中的深度线
- 补给积分条以“小型附属组件”形式贴在棋盘上方
- 深度数值牌以“小牌子”形式贴在深度线侧边

采矿区域规则补充：

- 当前可视采矿区域为 `7×6` 格。
- 采矿棋盘在中部主场景中必须横向居中摆放，作为视觉焦点。
- 在任意主流移动端竖屏尺寸下，棋盘、右侧工具条、底部背包与返场按钮必须在一屏内完整呈现，不依赖页面整体滚动。
- 采矿区域中需要加入一条深度线。
- 深度线的位置表示“当前采矿所在的最下方一行”和“倒数第二行”之间的位置。
- 采矿区域是否整体下移，取决于深度线的位置。
- 最下方一行需要始终保留，用于持续给玩家提供“下方仍可继续挖”的感知。
- 最下方一行必须展示真实的下一行矿井区域，而不是单独的提示层、占位层或伪造预览层。
- 棋盘内不应长期占用大块提示文案，如“从顶部入口开始挖”“路径连通后才能继续”等，这类规则应通过新手引导、轻量 toast 或首次提示完成，而不是压住主视觉。

### 3.5 底部操作区

考虑移动端单手操作和高频点击便利性，本局所有高频操作型道具统一放在底部操作区。

底部操作区固定展示本局可使用工具，并横向排列：

- 木镐子剩余数
- 横排炸弹
- 九宫格炸弹
- 背包快捷入口
- 收队返场按钮（固定在最右下角）

工具需要支持点击切换，并有清晰的剩余数量表现。

布局要求：

- 工具区位于屏幕底部安全操作区
- 工具可采用右侧纵向独立工具条，但整体仍需位于单手易点区域
- 数量需要与当前本局工具库存实时同步
- 当前选中工具必须有明显高亮态
- `收队返场` 固定放在最右下角，与常用工具按钮拉开间距，避免误触
- `收队返场` 不与镐子、炸弹按钮混排在连续点击热区中，应作为独立的退出操作按钮处理
- 背包可保留在底部独立按钮位，与右侧工具条形成职责分离

### 3.6 顶部收获展示区

本局收获展示从底部调整至顶部信息区，仅用于承接“收益累计感知”，不承载操作功能。

展示内容：

- 本局已获得的金币
- 本局已获得的矿品 / 矿石
- 本局已获得的材料

设计要求：

- 顶部收获展示仅用于查看累计结果，不可作为高频点击入口
- 资源增长时可做轻量数字跳动或飘字反馈，但不弹出阻断式窗口
- 背包入口保留在底部操作区，避免与顶部展示区产生交互职责混淆
- 宝箱作为过程型掉落，不在顶部单独占展示位；开启后的内容直接计入对应资源或工具数量
- 顶部收获展示应作为飞入特效的目标锚点，金币、矿石、材料三个槽位需各自可接收对应资源飞入

### 3.7 提前退出与本局收获弹窗

无尽矿井需要提供主动结束入口，建议文案使用：`收队返场`。

#### 3.7.1 提前退出入口

```text
小游戏底部操作区最右下角独立按钮：[收队返场]
```

按钮需始终可见，保证玩家任意时刻都能主动结束本局。
同时需要通过独立位置、间距和视觉样式降低误触概率。

#### 3.7.2 结算流转

```text
[无尽矿井进行中]
    ↓
[点击「收队返场」]
    → 当前收益停止增长
    → 汇总本局已获得资源
    → 弹出「本局收获」弹窗
    ↓
[查看本局深度和收成]
    ↓
[返回矿区主页]
```

### 3.8 移动端布局约束

本玩法以竖屏移动端为唯一主设计目标，布局必须以“单屏完成、棋盘优先”为原则。

明确要求：

- 首屏不允许依赖页面整体上下滚动查看核心玩法区域。
- 在常见移动端竖屏尺寸下，玩家进入页面后必须同时看到：
- 顶部 HUD
- 状态信息区
- 顶部收获累计区
- 挖掘棋盘主体
- 右侧工具条
- 背包入口
- 收队返场
- 为保证单屏展示，可以压缩非核心区域高度，但不得压缩棋盘到难以点击。
- 优先缩减的区域依次为：
- 装饰性矿车 / 立牌
- 顶部信息卡高度
- 收获区留白
- 工具按钮外边距
- 不允许以缩小字体到难以识别的方式换取单屏；需优先通过栅格、留白和装饰裁剪解决。

## 4. 资源模型

### 4.1 体力规则

- 消耗 `1` 点体力可以下矿井玩 `1` 局
- 体力属于主矿场通用资源
- 体力不足时不可进入无尽矿井
- 本模式不额外创造门票或独立入场资源

### 4.2 镐子规则

- 每一局固定给 `20` 个木镐子用于探索
- 木镐子是本局基础探索资源，不继承上局剩余数量
- 点击一个可破坏砖块，消耗 `1` 个木镐子并造成 `1` 点伤害
- 点击不可敲砖不消耗木镐子
- 局内可通过积分 bar、工具格和宝箱获得额外工具，但开局固定木镐子数量始终为 `20`
- 积分 bar 的基础积分来源为“层数加深 1 层，积分 +1”
- 局内掉落内容也可以直接提供额外积分奖励 `+n`

### 4.3 炸弹规则

- 横排炸弹和九宫格炸弹属于高价值范围工具
- 炸弹可来自局中补给
- 炸弹不计入“固定 20 个木镐子”的基础配置中
- 炸弹只影响可破坏砖块，不影响不可敲砖

### 4.4 收益与背包规则

本模式收益必须与主矿场资源模型保持一致。

| 资源 | 局内表现 | 结算处理 | 去向 |
| --- | --- | --- | --- |
| 矿品 | 破坏矿砖后掉落 | 结算汇总 | 进入主背包/主仓库 |
| 金币 | 破坏砖块、宝箱获得 | 结算汇总 | 进入主钱包 |
| 材料 | 如沙晶、铁矿 | 结算汇总 | 进入主材料背包 |
| 积分 | 通过加深层数或积分掉落获得 | 即时累计到积分 bar | 用于当前深度类型的积分奖励 |
| 宝箱 | 挖到后自动开启 | 奖励即时叠加 | 工具留在本局，矿石进入本局收获 |
| 工具补给 | 木镐子、炸弹等 | 即时获得 | 仅本局继续推进使用 |

关键约束：

- 矿品价值统一参考以下矿品表：

| 矿石名称 | 稀有等级 |
| --- | --- |
| 普通石 | 1级·普通 |
| 萤石 | 2级·入门宝石 |
| 青金石 | 3级·稀有 |
| 紫晶 | 4级·珍稀 |
| 蓝晶 | 5级·高阶晶石 |
| 玉石 | 6级·贵重 |
| 火曜石 | 7级·活动稀有 |
| 陨铁 | 8级·传说 |
| 星辉石 | 9级·神话 |

- 不在本模式内重新定义矿品售价
- 模式只影响矿品的出现概率、数量和深度分布
- 稀有矿石、红砖盲盒矿和宝箱均由深度阈值控制解锁与概率提升
- 挖到宝箱后，宝箱自动开启，其中开出的矿石和工具数量直接叠加到当前本局收获与当前工具数量中
- 宝箱自动开启后，矿石和工具可以同时出现
- 掉落内容本身可以包含积分奖励，直接累加到当前积分 bar
- 矿石等级按层数逐步解锁：
- `1` 层可出现 `1级·普通` 和 `2级·入门宝石`
- `2` 层可出现 `3级·稀有` 和 `4级·珍稀`
- 后续按相同规则递进，即更深层逐步开放更高等级矿石

## 5. 核心循环

### 5.1 完整流程

```text
矿区主页
    ↓
点击无尽矿井入口
    ↓
扣除1点体力
    ↓
系统发放本局固定20个木镐子
    ↓
点击砖块 / 使用炸弹 / 获取掉落
    ↓
向更深层推进，收益倍率逐步提升
    ↓
镐子与炸弹接近耗尽
    ↓
选择：继续下挖 / 主动止盈
    ↓
结算本局深度与收益
    ↓
资源回流主矿场
```

### 5.2 核心乐趣拆解

- 开局简单：1 点体力入场，固定 20 个木镐子，规则一眼能懂
- 挖掘反馈：点击即破坏、掉落即时飞出
- 收益预期：越深层越值钱
- 续航判断：如何利用剩余工具和局中补给走得更深
- 止盈抉择：现在收手还是再冲一层

## 6. 格子与砖块类型

### 6.1 核心砖块与遮挡层

本模式中的格子采用“遮挡层 + 底层真实内容”的结构，而不是所有砖块都直接裸露显示。

| 名称 | 耐久 | 说明 |
| --- | --- | --- |
| 深红色墙面 | 1 | 统一遮挡层，覆盖在大部分格子表面，敲开后才显示下面真实内容 |
| 浅红色沙砖 | 1 | 基础砖块，稳定低收益 |
| 石砖矿 | 2 | 高价值砖块，产出核心矿品 |
| 红砖盲盒矿 | 1 | 惊喜砖块，可能给宝箱或工具补给 |
| 不可敲砖 | 无限 | 障碍砖块，不可被普通挖掘破坏 |
| 宝箱 | 1 次发现 | 可直接出现在格子中，也可能在墙面下被发现，发现后自动开启 |
| 工具格 | 1 次发现 | 以 `🛠️` 形式出现在砖的正中间，相当于透视提示该格内有工具 |

### 6.2 砖块交互说明

| 内容 | 点击结果 | 掉落定位 |
| --- | --- | --- |
| 深红色墙面 | 敲击 1 次后碎掉，浮现下方真实内容 | 无直接收益 |
| 浅红色沙砖 | 直接破坏 | 基础金币、基础材料 |
| 石砖矿 | 先裂开再破坏 | 主矿场矿品、材料、少量炸弹 |
| 红砖盲盒矿 | 直接破坏 | 宝箱、工具补给、高波动收益 |
| 不可敲砖 | 无效反馈 | 无 |
| 宝箱 | 被挖到后自动开启 | 开出矿石或工具，并即时叠加 |
| 工具格 `🛠️` | 被挖到后直接获得工具 | 木镐子、炸弹等工具补给 |

交互规则补充：

- 大部分格子默认先显示深红色墙面。
- 当玩家使用木镐子或炸弹打碎墙面后，才会浮现下面真实的砖类型、宝箱或空内容。
- 下层真实内容不一定是砖，也可能没有砖，而是宝箱。
- 工具格出现在砖的正中间，用于提前提示该格存在工具。
- 宝箱随机出现在格子中，表现上区别于普通砖块。

### 6.3 深度分层规则

| 深度段 | 设计重点 |
| --- | --- |
| 地表矿区 | `1-30` 层，沙砖多、障碍少，帮助玩家消化开局 20 个木镐子的价值 |
| 深地表 | `31-80` 层，开始出现宝箱，仍以基础收益为主，地图开始更复杂 |
| 浅海 | `81-150` 层，从第三个深度类型开始出现稀有矿石和红砖盲盒矿，首次出现稀有矿石时给用户 toast 提示 |
| 深海 | `151-260` 层，红砖盲盒矿和宝箱在该深度类型内出现概率继续提升 |
| 地心火山泥 | `261+` 层，高价值资源最集中，但障碍、遮挡层和高耐久砖占比最高 |

深度分层的明确规则：

- 每个深度类型包含的层数不是等长的，而是随着深度加深逐步变大。
- 越靠后的深度类型，覆盖的层数越多，单一深度类型内的挑战持续时间越长。
- 深度类型层数区间按常见增长游戏节奏设置：前期切换更快、中期开始拉长、后期显著拉长，让玩家更快进入新鲜内容，再逐步延长高价值区域停留时间。
- 稀有矿石从第三个深度类型开始出现。
- 稀有矿石首次出现时，需要给玩家一个 toast 提示。
- 宝箱从第二个深度类型开始出现。
- 红砖盲盒矿从第三个深度类型开始出现。
- 红砖盲盒矿和宝箱在进入对应深度类型后即具有基础出现概率。
- 红砖盲盒矿和宝箱在每一个深度类型中，都会随着深度继续加深而逐步提高出现概率。
- 各类内容的具体出现层数、概率曲线和阈值由数值表配置，本 PRD 明确规则，不在正文写死数值。

## 7. 深度推进规则

### 7.1 单局结构

无尽矿井采用持续向下生成结构：

- 地图宽度固定为 `7` 格
- 地图高度固定为 `6` 格的可视区域
- 地图向下无限生成
- 画面只展示当前附近可视区域
- 玩家通过不断清理砖块获得更深位置

### 7.2 阶段信息

收益阶段按深度类型和阶段节点共同划分，不要求所有阶段等长。

阶段信息区至少展示：

- 当前深度
- 当前深度类型
- 当前深度类型积分进度
- 下一阶段提示

补充规则：

- 深度类型的层数跨度随深度递增。
- 同一深度类型内部可继续细分多个阶段节点，用于承接倍率变化和内容递进。

### 7.3 阶段收益变化

深度阶段提升后，增加以下内容：

- 高价值矿砖占比
- 矿品数量倍率
- 盲盒矿收益上限
- 宝箱价值期望
- 稀有矿石出现概率
- 红砖盲盒矿出现概率
- 进度条积分积满所需敲击次数
- 进度条积分奖励丰富度
- 砖块地图生成难度

具体数值跟随主数值表，不在本 PRD 中写死。

阶段收益变化的明确规则：

1. 稀有矿石从第三个深度类型开始解锁；后续阶段越深，稀有矿石出现概率越高。
2. 红砖盲盒矿从第三个深度类型开始解锁；进入对应深度类型后即具备基础概率，后续阶段越深，出现概率越高。
3. 宝箱从第二个深度类型开始解锁；进入对应深度类型后即具备基础概率，后续阶段越深，出现概率越高。
4. 进度条积分的积满门槛会随着深度增加而变大，需要更多敲击次数才能积满。
5. 进度条积分奖励会随着深度类型提升而变得更加丰富，奖励内容包含 `炸弹*n`、`木镐子*n`。
6. 随着深度加深，挖矿难度需要同步加大，具体体现在砖块地图生成分布上，包括但不限于深色砖占比提升、不可敲砖占比提升、高价值砖块更分散、可直线下挖路径减少。
7. `7×6` 地图砖块分布规则按常见增长游戏节奏设置：前期保证可理解和连续破坏感，中期增强路径选择，后期增强阻挡、绕路和高价值目标的获取成本。
8. 地图生成必须永远保证至少存在一条可继续向下挖的路径，避免玩家因纯随机分布被完全封死。

### 7.4 工具耗尽规则

当本局木镐子和炸弹全部耗尽时：

- 本局立即结束
- 进入本局收获结算弹窗
- 本次挑战收益停止增长

### 7.5 玩家结束选择

玩家可在以下时机结束本局：

- 任意时刻点击 `收队返场` 主动退出
- 木镐子和炸弹全部耗尽

## 8. 工具系统

### 8.1 工具列表

| 工具 | 作用 | 来源 |
| --- | --- | --- |
| 木镐子 | 单体基础探索 | 每局固定 20 个开局 + 局中补给 |
| 横排炸弹 | 清理整行 | 局中补给 |
| 九宫格炸弹 | 清理局部范围 | 局中补给 |

### 8.2 木镐子

- 木镐子是本模式最核心的基础资源
- 每局固定发放 `20` 个
- 该规则必须作为模式记忆点长期稳定

### 8.3 横排炸弹

- 直接破坏目标所在整行的可破坏砖块
- 不破坏不可敲砖
- 用于快速清障和追矿

### 8.4 九宫格炸弹

- 直接破坏目标周围九宫格内的可破坏砖块
- 边界处只结算有效格子
- 用于集中清收益区

### 8.5 工具补给说明

首发建议：

- 工具补给来源固定为 3 种：
- 积分 bar 积满后，获得固定补给工具
- 挖到工具格 `🛠️` 后，直接获得工具
- 挖到宝箱 `🪎` 后，自动开启，开出工具和矿石中的一种或多种组合

- 木镐子补给主要用于延长基础探索
- 炸弹补给主要用于关键时刻突破或追收益
- 积分 bar 的积满所需敲击次数会随着深度逐步变大
- 积分 bar 积满后的奖励会随着深度类型提升而更丰富
- 局内掉落本身也可以提供积分奖励，直接增加当前积分 bar 进度
- 每个深度类型中，都可能随机掉落当前深度类型最大层数 `30%` 的积分奖励
- 工具格当前只产出 3 种工具：木镐子、横排炸弹、九宫格炸弹

这样可以保持规则简单，并凸显固定 20 个木镐子的基础节奏。

## 10. 视觉反馈与动画清单

### 10.1 环境表现

- 深度加深时背景色和矿层纹理变化
- 灯光、粉尘和矿井支撑结构变化
- 向下推进时有镜头位移感
- 不同深度类型有明确背景区别：
- 地表矿区
- 深地表
- 浅海
- 深海
- 地心火山泥

### 10.2 挖掘反馈

- 点击砖块时播放敲击反馈
- 破坏成功时砖块碎裂并掉落收益
- 使用炸弹时播放范围爆破反馈
- 木镐点击时，应有单点聚焦高亮或命中闪光
- 横排炸弹和九宫格炸弹释放前后，都应有明确的范围高亮反馈，让玩家直观看到将被波及的区域
- 爆炸范围反馈需要直接贴在命中的格子上，而不是只在棋盘外播放通用爆炸特效
- 资源掉落后，应有“矿石 / 金币 / 材料”从命中格子飞入顶部累计区的飞入特效
- 顶部累计区在资源飞入命中时，需同步做轻量跳动或闪光反馈
- 炸开形成的空层需要具备明显的下陷坑道感，表现为：
- 中心更暗
- 边缘内阴影更重
- 与未敲通格子的体块感形成对比
- 未敲通的格子需要具备上浮土层感，表现为：
- 顶部高光更强
- 底部有厚边与投影
- 边缘轮廓可略微不规则，模拟天然泥土层

### 10.3 高价值收益反馈

- 矿砖命中时加强音效
- 盲盒矿掉宝箱时强化弹出反馈
- 进入新阶段时做轻量庆祝表现
- 解锁稀有矿石、红砖盲盒矿、宝箱时应有首次出现提示反馈
- 宝箱自动开启时，应有奖励展开并即时叠加反馈

## 11. 交互逻辑

### 11.1 点击砖块挖掘

- 点击可破坏砖块时消耗 `1` 个木镐子并造成 `1` 点伤害
- 点击不可敲砖时只给无效反馈，不消耗木镐子

### 11.2 炸弹交互

- 先点击炸弹图标，再点击目标区域释放
- 释放前显示范围预览
- 若目标不可释放，则给出无效提示
- 横排炸弹的范围预览需覆盖当前目标所在整行
- 九宫格炸弹的范围预览需覆盖目标周围 `3×3` 区域
- 范围预览与实际爆炸命中区域必须完全一致，避免误导玩家

### 11.3 结算交互

- 点击 `收队返场` 后直接进入结算弹窗
- 工具耗尽后直接进入结算弹窗

### 11.4 继续下挖交互

- 清理当前区域后，画面自动向下补充新砖块
- 达到更深阶段时弹出阶段提示

## 12. 边界与异常处理

- 体力不足时，入口置灰并提示 `体力不足`
- 主背包满时，需按主系统规则处理，不在本模式内单独兜底
- 结算中断线时，按服务端已记录收益补发
- 玩家退出游戏后，未结算收益按主系统规则恢复或结算

## 13. P0 MVP 范围

### 13.1 P0 必须实现

- `1` 点体力 = `1` 局
- 每局固定 `20` 个木镐子
- 5 类砖块
- 3 类工具
- 深度持续生成
- 退出机制仅包含主动退出和工具耗尽
- 局中补给来源为积分 bar、工具格、宝箱
- 稀有矿石、红砖盲盒矿、宝箱按深度阈值逐步解锁
- 稀有矿石和红砖盲盒矿的出现概率随深度提升
- 积分 bar 门槛随深度提升，奖励随深度类型变丰富
- 深度类型至少区分地表矿区、深地表、浅海、深海、地心火山泥
- 深度加深后砖块地图分布难度提升
- 工具耗尽直接结算
- 收益统一进入主矿场资源体系
- 矿品价值沿用既有矿品表
- 移动端单屏布局
- 顶部收获区飞入特效
- 炸弹范围预览 / 命中特效
- 深度线侧边深度牌
- 挖空通路的下陷表现
- 未敲通土层的上浮表现

### 13.2 P0 暂不包含

- 复杂局内商店系统
- 特殊事件层
- 路线分支层
- 多种失败惩罚
- 独立成就体系

## 14. 验证重点

### 14.1 玩家是否理解单局成本

重点验证玩家是否能快速理解：

- 进入一次要消耗 1 点体力
- 每局开局固定只有 20 个木镐子

### 14.2 玩家是否理解深挖收益

重点验证玩家是否能感知：

- 越深层越值钱
- 同样 1 个木镐子在深层更有收益价值
- 达到更深层后会逐步解锁稀有矿石、红砖盲盒矿和宝箱
- 进度条奖励虽然更难积满，但奖励也更丰富
- 地图虽然更难，但始终存在至少一条可继续下挖路径

### 14.3 工具耗尽后的结束反馈是否成立

重点验证玩家是否会在：

- 主动止盈
- 工具耗尽结算

之间形成清晰认知，而不是对结束时机感到困惑。

### 14.4 单局时长是否适合移动端

重点验证：

- 固定 20 个木镐子开局是否让节奏足够紧凑
- 在当前固定资源规则下是否仍适合竖屏移动端短时体验

## 15. 一句话总结

无尽矿井是矿场增长游戏中的主动下挖模式：玩家消耗 `1` 点体力进入一局，系统固定发放 `20` 个木镐子作为基础探索资源，在深度推进和收益止盈的选择中，把本局产出稳定回流到主矿场资源体系。
