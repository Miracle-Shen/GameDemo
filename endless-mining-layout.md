# 无尽挖矿移动端布局示意图

> 适用场景：竖屏移动端小游戏 / 休闲挖矿玩法 / 无尽下探关卡  
> 当前版本：去掉底部「矿车 / 背包 / 自动挖掘」操作区，并在「深度线」下方补充一行可挖层。

## 2. 调整说明

本版布局相较原始方案做了以下调整：

- 去掉底部固定操作区：不再展示「矿车 / 背包 / 自动挖掘」。
- 主玩法区获得更多纵向空间。
- 在「深度线」下方增加一行可挖矿层。
- 强化「无尽向下挖掘」的连续感。
- 底部区域直接承接下一层矿块，不打断玩家视线。

## 4. 移动端布局分区

| 区域 | 位置 | 功能说明 |
|---|---|---|
| 顶部资源栏 | 屏幕最上方 | 展示等级、体力、金币、钻石等长期资源 |
| 标题栏 | 资源栏下方 | 当前玩法名称、帮助入口、关闭按钮 |
| 进度奖励条 | 标题栏下方 | 展示当前矿层目标进度，如 `41/55` |
| 左侧工具栏 | 主玩法区左侧 | 放置镐子、炸药、雷管等可消耗工具 |
| 中央矿井区 | 屏幕核心区域 | 玩家主要交互区域，展示土层、矿石、障碍、角色、深度线和下一行可挖层 |
| 右侧任务入口 | 主玩法区右侧 | 展示限时任务、双倍奖励、倒计时 |
| 安全区 | 顶部 / 底部 | 适配刘海屏、灵动岛、手势条 |

## 6. 玩法元素说明

### 6.1 矿层元素

| 元素 | 表现 | 作用 |
|---|---|---|
| 普通土层 | 黄色 / 棕黄色块状区域 | 基础可挖区域 |
| 岩石层 | 深灰色块状区域 | 挖掘难度更高 |
| 金矿 | 橙黄色晶体 | 高价值资源 |
| 银矿 / 白矿 | 白色晶体 | 中等价值资源 |
| 阻挡砖块 | 红色砖块 | 需要特殊工具或多次挖掘 |
| 空洞区域 | 透明 / 深色背景 | 已挖开的路径 |
| 下一行可挖层 | 位于深度线下方 | 暗示矿井继续向下延展，强化无尽挖掘感 |

### 6.2 工具栏元素

```text
┌────────────┐
│  ⛏ 镐子     │
│  数量 125   │
├────────────┤
│  🧨 炸药     │
│  数量 337   │
├────────────┤
│  📦 雷管组   │
│  数量 75    │
└────────────┘
```

| 工具 | 用途 |
|---|---|
| 镐子 | 单格挖掘，基础消耗 |
| 炸药 | 范围爆破，适合清理密集土层 |
| 雷管组 | 大范围爆破或指定方向爆破 |

## 9. 信息流与交互流

```text
玩家进入矿井
    ↓
查看当前矿层目标 41/55
    ↓
选择工具：镐子 / 炸药 / 雷管
    ↓
点击矿层进行挖掘
    ↓
获得矿石 / 金币 / 特殊奖励
    ↓
深度线下移或矿层向上滚动
    ↓
露出下一行可挖层
    ↓
触发阶段奖励 / 限时双倍 / 深度目标
    ↓
继续无尽挖掘
```

## 10. 视觉风格建议

| 项目 | 建议 |
|---|---|
| 整体风格 | 卡通休闲、圆角、高饱和度 |
| 主色调 | 土黄色、矿洞棕、金币橙 |
| 辅助色 | 蓝色 UI 面板、绿色进度条、红色炸药 |
| 反馈效果 | 碎石飞溅、金币弹出、进度条增长 |
| 动效重点 | 挖掘敲击、爆破扩散、矿石收集、深度推进 |
| 移动端适配 | 重点按钮尺寸不小于 44px，底部避开手势区 |

---

## 11. 当前布局重点

- 中央大面积矿井区突出主要操作。
- 左侧工具栏方便快速切换道具。
- 右侧任务入口承载限时奖励与商业化激励。
- 顶部资源栏 + 进度条强化长期成长目标。
- 底部不再放固定操作按钮，让矿井区域占满更多空间。
- 深度线下方露出下一行可挖层，更适合表现「无尽向下挖掘」。
