# 《无限矿井》无尽模式需求文档

本文档描述矿场增长游戏中的扩展玩法模式“无尽矿井”。该模式是主矿场系统中的主动操作玩法入口。玩家消耗体力进入矿井，在固定工具配额下向下探索，通过挖掘砖块获得矿品、货币、材料和宝箱收益，并将产出回流到主矿场资源体系。

版本：v1.2 | 创建日期：2026-06-18 | 最后更新：2026-06-22

## 1. 系统定位

### 1.1 挖矿小游戏在游戏中的角色

无尽矿井是矿场增长游戏中的主动操作玩法，是主矿场挂机增长之外的额外收益入口。

玩家从矿区主页进入无尽矿井，消耗 `1` 点体力开启 `1` 局挑战。进入后，系统固定发放 `20` 个木镐子、`1` 个横排炸弹和 `1` 个九宫格炸弹作为本局初始探索资源，玩家通过点击砖块和使用范围工具向下推进，在有限基础资源条件下争取更深层数和更高收益。

核心链路：

```text
矿区主页 → 点击无尽矿井入口 → 消耗1点体力入场
    ↓
系统发放本局初始工具：20个木镐子、1个横排炸弹、1个九宫格炸弹
    ↓
点击砖块 / 使用炸弹 / 获取掉落
    ↓
继续向下推进并提升深度阶段
    ↓
工具耗尽
    ↓
主动退出或工具耗尽 → 本局结算 → 收益回流主矿场
```

### 1.2 设计目标

| 目标 | 说明 |
| --- | --- |
| 模式身份清晰 | 这是主矿场内的扩展模式，不独立产生新经济体系 |
| 开局成本简单 | 1 点体力进入 1 局，单局固定 20 个木镐子 + 2 个初始炸弹，规则直接易懂 |
| 深挖收益明确 | 越深层收益期望越高，让玩家有持续推进动力 |
| 决策张力成立 | 玩家需要在继续下挖和及时止盈之间做判断 |
| 操作边界清晰 | 玩家必须清楚什么格子现在能挖、什么位置可以放炸弹 |
| 资源模型统一 | 所有收益统一进入主矿场背包、钱包和材料系统 |

### 1.3 与矿区系统的关系

```text
矿区主页 → 点击「无尽矿井」→ 无尽矿井玩法（本页面）
         ↘ 点击其他建筑/系统 → 原有矿区玩法
```

无尽矿井沿用主矿区已确定的系统口径：

- 顶部继续展示体力、主货币、付费货币等主系统信息
- 消耗体力规则与主矿区一致
- 收益统一进入主背包和主货币体系
- 不单独创建模式专属售卖价格体系
- 矿品价值统一参考既有矿品表

## 2. 玩法总览

### 2.1 一句话玩法定义

无尽矿井是一种“消耗 1 点体力进入、每局固定 20 个木镐子和两种炸弹各 1 个开局、通过深度推进争取更高收益”的下挖式挖矿玩法。

### 2.2 单局与深度推进的关系

- `1` 点体力 = `1` 局无尽矿井
- `1` 局无尽矿井 = 以固定 `20` 个木镐子、`1` 个横排炸弹、`1` 个九宫格炸弹开局的一次连续下挖挑战
- 深度推进 = 本局中持续向更深区域探索并提升收益阶段

也就是说，体力限制开局次数，镐子定义本局基础探索次数，深度决定本局收益上限。

### 2.3 本次功能澄清要点

为避免玩家和实现侧对“什么能挖、什么能炸”理解不一致，本模式新增明确口径：

1. 木镐只能点击“与当前坑道连通”的格。
2. 深红遮挡层下未显露的格，不允许跳过遮挡层直接点击破坏，必须先破坏深红遮挡层，才能继续破坏下面的砖。
3. 炸弹可以对“未连通但可见”的格产生波及效果，但炸弹的落点必须合法。
4. 炸弹落点的“合法”定义为：必须在“当前可连通的坑道上”，不能直接把炸弹放在不可连通的封闭格、隐藏格或未接通区域上。
5. 最底行作为真实可交互行，规则与其他 `7×6` 可视区域完全一致。

- 深度信息贴近棋盘中的深度线展示。
- `补给积分` 展示在挖掘区域正上方，作为轻量附属状态条。
- 所有高频工具按钮统一收纳在底部操作区，作为主操作入口。

### 3.2 顶部 HUD

顶部 HUD 延续矿区主页的设计，不单独创造新口径。

| 元素 | 说明 |
| --- | --- |
| 返回按钮 | 左上返回入口，点击后弹出确认框 |
| 体力 | 展示当前剩余体力，进入本模式时消耗 |
| 金币 | 主系统通用货币 |
| 玉币 | 主系统付费货币 |
| 设置 | 打开设置弹层 |

### 3.3 深度与目标信息区

该区域是本玩法的信息锚点，玩家进入后应立即看到当前推进状态。

| 元素 | 说明 |
| --- | --- |
| 当前深度类型 | 展示当前所处深度类型，如地表矿区、深地表、浅海、深海、地心火山泥 |
| 本局推进 | 展示当前深度类型下的补给积分进度，如 `4/12` |
| 本局状态 | 展示如 `推进中`、`工具耗尽`、`结算中`、`确认返场` 等短状态文案 |
| 本局收获展示 | 展示本局累计获得的金币、矿石、材料等资源，仅供查看，不承载点击操作 |

布局约束：

- 顶部信息区承载“状态信息 + 本局收获累计展示”。
- 顶部区域不放置高频点击道具按钮。
- 玩家在挖掘过程中应能持续看到本局收益增长，但不需要在顶部进行任何工具操作。
- 宝箱不在顶部单独累计展示；宝箱开启后，开出的矿石、材料、金币或工具直接并入对应展示位。

### 3.4 中部矿井主场景

中部为“矿井环境 + 可交互砖块区域”的组合场景。

场景表现应包含：

- 岩层和矿层变化
- 木制支撑结构
- 灯光与尘土粒子
- 不同深度类型的环境色、背景元素和氛围变化
- 下挖推进的视觉位移感
- 采矿区域中的深度线
- 补给积分条以“小型附属组件”形式贴在棋盘上方
- 深度数值牌以“小牌子”形式贴在深度线侧边

采矿区域规则补充：

- 当前可视采矿区域为 `7×6` 格。
- 棋盘最下方保留一整行真实可交互区域，用于持续传达“下方仍可继续挖”。
- 最下方一行展示真实下一行矿井区域。
- 最下方一行在规则上与其余可视区域完全一致。

### 3.5 底部操作区

考虑移动端单手操作和高频点击便利性，本局所有高频操作型道具统一放在底部操作区。

底部操作区固定展示本局可使用工具，并横向排列：

- 木镐子剩余数
- 横排炸弹
- 九宫格炸弹
- 背包快捷入口
- 收队返场按钮

工具条要求：

- 只保留必要信息：图标、剩余数量、选中态
- 不在工具条中加入额外的说明性文字、规则说明或长文案
- 所有工具按钮统一集中在底部操作区
- 工具需要支持点击切换，并有清晰的剩余数量表现

布局要求：

- 工具区位于屏幕底部安全操作区
- 所有工具入口必须保持在底部
- 当前选中工具必须有明显高亮态
- `收队返场` 应作为独立退出操作按钮处理，避免误触

### 3.6 顶部收获展示区

本局收获展示位于顶部信息区，用于承接“收益累计感知”，不承载操作功能。

展示内容：

- 本局已获得的金币
- 本局已获得的矿品 / 矿石
- 本局已获得的材料

设计要求：

- 顶部收获展示仅用于查看累计结果，不可作为高频点击入口
- 资源增长时可做轻量数字跳动或飘字反馈，但不弹出阻断式窗口
- 宝箱作为过程型掉落，不在顶部单独占展示位；开启后的内容直接计入对应资源或工具数量
- 顶部收获展示应作为飞入特效的目标锚点

### 3.7 提前退出与本局收获弹窗

无尽矿井需要提供主动结束入口，建议文案使用：`收队返场`。

#### 3.7.1 提前退出入口

```text
小游戏底部操作区最右下角独立按钮：[收队返场]
```

按钮需始终可见，保证玩家任意时刻都能主动结束本局。

#### 3.7.2 结算流转

```text
[无尽矿井进行中]
    ↓
[点击「收队返场」]
    → 弹出确认框
    → 点击确认返场
    → 当前收益停止增长
    → 汇总本局已获得资源
    → 弹出「本局收获」弹窗
    ↓
[查看本局深度和收成]
    ↓
[返回矿区主页]
```

### 3.8 移动端布局约束

本玩法以竖屏移动端为唯一主设计目标，布局必须以“单屏完成、棋盘优先”为原则。

明确要求：

- 首屏以单屏完整呈现核心玩法区域为目标。
- 在常见移动端竖屏尺寸下，玩家进入页面后必须同时看到顶部 HUD、状态信息区、顶部收获累计区、挖掘棋盘主体、底部工具条、背包入口、收队返场。
- 为保证单屏展示，可以压缩非核心区域高度，但不得压缩棋盘到难以点击。

## 4. 资源模型

### 4.1 体力规则

- 消耗 `1` 点体力可以下矿井玩 `1` 局
- 体力属于主矿场通用资源
- 体力不足时不可进入无尽矿井
- 本模式不额外创造门票或独立入场资源

### 4.2 镐子规则

- 每一局固定给 `20` 个木镐子用于探索
- 木镐子是本局基础探索资源，不继承上局剩余数量
- 点击一个可破坏砖块，消耗 `1` 个木镐子并造成 `1` 点伤害
- 点击不可敲砖不消耗木镐子
- 局内可通过积分 bar、工具格和宝箱获得额外工具，但开局固定木镐子数量始终为 `20`
- 积分 bar 的基础积分来源为“层数加深 1 层，积分 +1”
- 局内掉落内容也可以直接提供额外积分奖励 `+n`

### 4.3 炸弹规则

- 横排炸弹和九宫格炸弹属于高价值范围工具
- 每局开局固定发放 `1` 个横排炸弹和 `1` 个九宫格炸弹
- 炸弹也可来自局中补给
- 炸弹只影响可破坏砖块，不影响不可敲砖
- 炸弹的波及范围可以覆盖未连通但可见的格
- 炸弹的施放落点必须合法
- 炸弹落点的合法定义为：必须在当前“可连通的坑道上”
- 炸弹仅可投放在当前可连通坑道中的合法操作格上

### 4.4 收益与背包规则

本模式收益必须与主矿场资源模型保持一致。

单局收益来源统一收敛为 4 类：

1. 挖到矿石 / 挖到工具
2. 积分 bar 的奖励
3. 局中随机掉落的积分和金币
4. 挖到宝箱并自动开启后的矿石或工具

补充说明：

- 局中随机掉落的积分和金币，在特效播放后应直接完成对应数值增长，无需玩家额外点击拾取。
- 宝箱只作为 UI 展示物和过程反馈物，不单独作为收益实体统计；真正计入本局收益的是宝箱开启后得到的矿石或工具。
- 浅红色沙砖不属于收益来源，不负责提供基础金币或基础材料。

| 资源 | 局内表现 | 结算处理 | 去向 |
| --- | --- | --- | --- |
| 矿品 | 破坏矿砖后掉落 | 结算汇总 | 进入主背包/主仓库 |
| 金币 | 破坏砖块、宝箱获得 | 结算汇总 | 进入主钱包 |
| 材料 | 如沙晶、铁矿 | 结算汇总 | 进入主材料背包 |
| 积分 | 通过加深层数或积分掉落获得 | 即时累计到积分 bar | 用于当前深度类型的积分奖励 |
| 宝箱 | 挖到后自动开启 | 奖励即时叠加 | 工具留在本局，矿石进入本局收获 |
| 工具补给 | 木镐子、炸弹等 | 即时获得 | 仅本局继续推进使用 |

关键约束：

- 不在本模式内重新定义矿品售价
- 模式只影响矿品的出现概率、数量和深度分布
- 稀有矿石、红砖盲盒矿和宝箱均由深度阈值控制解锁与概率提升
- 宝箱自动开启后，矿石和工具可以同时出现

## 5. 核心循环

### 5.1 完整流程

```text
矿区主页
    ↓
点击无尽矿井入口
    ↓
扣除1点体力
    ↓
系统发放本局初始工具：20个木镐子、1个横排炸弹、1个九宫格炸弹
    ↓
点击砖块 / 使用炸弹 / 获取掉落
    ↓
向更深层推进，收益倍率逐步提升
    ↓
镐子与炸弹接近耗尽
    ↓
选择：继续下挖 / 主动止盈
    ↓
结算本局深度与收益
    ↓
资源回流主矿场
```

### 5.2 核心乐趣拆解

- 开局简单：1 点体力入场，固定 20 个木镐子和两种炸弹各 1 个，规则一眼能懂
- 挖掘反馈：点击即破坏、掉落即时飞出
- 收益预期：越深层越值钱
- 路径判断：哪些格现在能挖、哪里能放炸弹，本身构成轻度策略
- 续航判断：如何利用剩余工具和局中补给走得更深
- 止盈抉择：现在收手还是再冲一层

## 6. 格子与砖块类型

### 6.1 核心砖块与遮挡层

本模式中的格子采用“遮挡层 + 底层真实内容”的结构。

为避免实现时对“砖”“路”“墙”的层级理解不一致，挖矿区统一定义为 3 层结构：

| 层级 | 名称 | 说明 |
| --- | --- | --- |
| 第一层（最底层） | 路 | 被挖通后的真实通路层，是玩家最终看到并继续向下推进的坑道基础层 |
| 第二层（中间层） | 地图内容层 | 地图主要生成层，承载浅红色沙砖、石砖矿、红砖盲盒矿、不可敲砖、宝箱、含工具格的砖块、空白格 |
| 第三层（最上层） | 深红色墙面 | 遮挡层，覆盖在第二层内容之上，需先被敲开后才能显露第二层真实内容 |

关于“路”的视觉定义补充：

- “路”的图片不需要边框
- “路”被挖开后，应直接露出当前深度类型对应的地表 / 坑道表面
- “路”的表现重点是“当前深度环境被挖通后的自然露出”
- 这样可以让玩家更直观感知：已经打通的区域与未打通区域属于两种明显不同的空间状态

第二层的具体内容定义如下：

- 浅红色沙砖
- 石砖矿
- 红砖盲盒矿
- 不可敲砖
- 宝箱
- 矿石格
- 工具格
- 空白格

其中，`空白格` 的定义为：第二层没有任何内容，直接露出第一层的路。

| 名称 | 耐久 | 说明 |
| --- | --- | --- |
| 深红色墙面 | 1 | 统一遮挡层，覆盖在部分格子表面，敲开后才显示下面真实内容；其占比会随深度推进逐步提升 |
| 浅红色沙砖 | 1 | 最易敲击、出现概率最高的基础砖块，用于构成主要路径与地图主体 |
| 石砖矿 | 2 | 高价值砖块，产出核心矿品 |
| 红砖盲盒矿 | 1 | 惊喜砖块，可能给宝箱或工具补给 |
| 不可敲砖 | 无限 | 障碍砖块，不可被普通挖掘破坏 |
| 宝箱 | 1 次发现 | 可直接出现在格子中，也可能在墙面下被发现，发现后自动开启 |
| 矿石格 | 1 次发现 | 用于提前提示该格内存在矿石 |
| 工具格 | 1 次发现 | 用于提前提示该格内有工具 |

### 6.2 砖块交互说明

| 内容 | 点击结果 | 掉落定位 |
| --- | --- | --- |
| 深红色墙面 | 敲击 1 次后碎掉，浮现下方真实内容 | 无直接收益 |
| 浅红色沙砖 | 直接破坏 | 不直接产生收益，仅用于打通路径和推进地图 |
| 石砖矿 | 先裂开再破坏 | 主矿场矿品、材料、少量炸弹 |
| 红砖盲盒矿 | 直接破坏 | 宝箱、工具补给、高波动收益 |
| 不可敲砖 | 无效反馈 | 无 |
| 宝箱 | 被挖到后自动开启 | 开出矿石或工具，并即时叠加 |
| 矿石格 | 被挖到后显示并获得对应矿石收益 | 用于提前提示该格存在矿石 |
| 工具格 | 被挖到后直接获得工具 | 木镐子、炸弹等工具补给 |

交互规则补充：

- 刚进入游戏时，深红色墙面的覆盖比例较低。
- 当玩家使用木镐子或炸弹打碎墙面后，才会浮现下面真实的砖类型、宝箱或空内容。
- 点击顺序遵循“先破坏遮挡层，再作用于第二层真实内容”。
- 浅红色沙砖不挂钩收益，不直接产生基础金币和基础材料。
- 第二层在生成地图时，必须始终保证存在向下挖掘的路径；不可敲砖不能把可继续推进的路径完全封死。
- 随着深度推进，深红色墙面的生成比例逐步提高。

### 6.3 深度分层规则

| 深度段 | 设计重点 |
| --- | --- |
| 地表矿区 | `1-30` 层，沙砖多、障碍少，帮助玩家消化开局工具组合的价值 |
| 深地表 | `31-80` 层，开始出现宝箱，仍以基础收益为主，地图开始更复杂 |
| 浅海 | `81-150` 层，从第三个深度类型开始出现稀有矿石和红砖盲盒矿，首次出现稀有矿石时给用户 toast 提示 |
| 深海 | `151-260` 层，红砖盲盒矿和宝箱在该深度类型内出现概率继续提升 |
| 地心火山泥 | `261+` 层，高价值资源最集中，但障碍、遮挡层和高耐久砖占比最高 |

深度分层的明确规则：

- 每个深度类型包含的层数随着深度加深逐步变大。
- 宝箱从第二个深度类型开始出现。
- 红砖盲盒矿和稀有矿石从第三个深度类型开始出现。
- 各类内容的具体出现层数、概率曲线和阈值由数值表配置，本 PRD 明确规则，不在正文写死数值。

## 7. 深度推进规则

### 7.1 单局结构

无尽矿井采用持续向下生成结构：

- 地图宽度固定为 `7` 格
- 地图高度固定为 `6` 格的可视区域
- 地图向下无限生成
- 画面只展示当前附近可视区域
- 玩家通过不断清理砖块获得更深位置

#### 7.1.1 P0 初始可视棋盘生成规则

P0 初始可视棋盘固定为 `7×6`，用于保证首屏手感一致。

首屏为已经存在一个小型起始坑道区域的状态：

- 首屏必须存在一个大约 `3~6` 个格子已经是“路”的挖矿区域
- 这片起始“路”区域用于帮助玩家立即理解：自己当前是从一段已打通的坑道继续往下挖
- 这片起始“路”区域必须与后续可继续挖掘的第一批目标格相邻，保证玩家进入后第一眼就知道哪里可以操作
- 首屏不应让玩家进入后出现“所有格子都能看见，但不知道第一下该点哪里”的理解成本

首屏模板要求：

- 地形层仍采用固定 `7×6` 模板
- 模板中至少保留一段明确的起始可连通坑道
- 首屏应保证至少存在 `1` 条可继续向下扩张的合法操作链
- 首屏可以出现少量矿砖、工具格或宝箱，但不应把高价值内容堆满，避免抢夺教学阶段的理解重点

### 7.2 阶段信息

收益阶段按深度类型和阶段节点共同划分，不要求所有阶段等长。

阶段信息区至少展示：

- 当前深度
- 当前深度类型
- 当前深度类型积分进度
- 下一阶段提示

### 7.3 阶段收益变化

深度阶段提升后，增加以下内容：

- 高价值矿砖占比
- 矿品数量倍率
- 盲盒矿收益上限
- 宝箱价值期望
- 稀有矿石出现概率
- 红砖盲盒矿出现概率
- 进度条积分积满所需敲击次数
- 进度条积分奖励丰富度
- 砖块地图生成难度

具体数值跟随主数值表，不在本 PRD 中写死。

#### 7.3.1 新行生成规则

每次“推进一层”时，在棋盘底部生成 `1` 行新数据，并删除顶部 `1` 行。

新行生成遵守：

1. 至少保留 `1` 条从当前底部可继续向下的可破坏路径
2. 浅红色沙砖是所有深度区中概率最高的内容
3. 深度越深：
   - 深红遮挡层出现概率更高
   - 不可敲砖占比更高
   - 矿砖分布更分散
   - 红砖盲盒矿、宝箱概率逐步提高
4. 宝箱仅在第 `2` 个深度区及以后可出现
5. 红砖盲盒矿与稀有矿石仅在第 `3` 个深度区及以后可出现
6. 每个深度区内的积分 bar 门槛会继续增长，补给越来越难积满

这条规则的产品含义是：

- 玩家必须始终感觉“还能继续往下挖”
- 但越往后，需要付出的路径成本、绕路成本和工具选择成本会越来越高

#### 7.3.2 深度区表

| 深度区编号 | 深度区名称 | 深度范围 | 对应背景资源 | 内容侧重点 |
|------------|-----------|------------|-------------------|------------------|
| 1 | 地表矿区 | `1-30` | `mine_surface_bg` | 沙砖多，障碍少，开局教学 |
| 2 | 深地表 | `31-80` | `mine_mid_bg` | 开始出现宝箱，路径略绕 |
| 3 | 浅海 | `81-150` | `mine_shallow_sea_bg` | 首次解锁稀有矿、盲盒矿 |
| 4 | 深海 | `151-260` | `mine_deep_sea_bg` | 宝箱与盲盒概率上升，墙面更密 |
| 5 | 地心火山泥 | `261+` | `mine_core_bg` | 高价值最高，遮挡和障碍最强 |

### 7.4 工具耗尽规则

当本局木镐子和炸弹全部耗尽时：

- 本局立即结束
- 进入本局收获结算弹窗
- 本次挑战收益停止增长

### 7.5 玩家结束选择

玩家可在以下时机结束本局：

- 任意时刻点击 `收队返场` 主动退出
- 木镐子和炸弹全部耗尽

## 8. 工具系统

### 8.1 工具列表

| 工具 | 作用 | 来源 |
| --- | --- | --- |
| 木镐子 | 单体基础探索 | 每局固定 20 个开局 + 局中补给 |
| 横排炸弹 | 清理整行 | 每局开局 1 个 + 局中补给 |
| 九宫格炸弹 | 清理局部范围 | 每局开局 1 个 + 局中补给 |

### 8.2 木镐子

- 木镐子是本模式最核心的基础资源
- 每局固定发放 `20` 个
- 该规则必须作为模式记忆点长期稳定

### 8.3 横排炸弹

- 每局开局固定发放 `1` 个
- 直接破坏目标所在整行的可破坏砖块
- 不破坏不可敲砖
- 用于快速清障和追矿

### 8.4 九宫格炸弹

- 每局开局固定发放 `1` 个
- 直接破坏目标周围九宫格内的可破坏砖块
- 边界处只结算有效格子
- 用于集中清收益区

### 8.5 工具补给说明

首发建议：

- 工具补给来源固定为 3 种：
- 积分 bar 积满后，获得固定补给工具
- 挖到工具格后，直接获得工具
- 挖到宝箱后，自动开启，开出工具和矿石中的一种或多种组合

- 木镐子补给主要用于延长基础探索
- 炸弹补给主要用于关键时刻突破或追收益

补给积分规则补充：

- 每推进 `1` 层：`supplyPoints +1`
- 掉落积分奖励时：立即增加 `supplyPoints`
- 当 `supplyPoints >= supplyTarget` 时：
  - 清空或扣除一轮目标值
  - 根据当前深度区发放一份工具补给
  - 触发轻量奖励反馈

推荐补给：

| 深度区 | 补给奖励 |
| --- | --- |
| 地表矿区 | 木镐子补给为主 |
| 深地表 | 木镐子 + 横排炸弹 |
| 浅海 | 木镐子 + 横排炸弹 / 九宫格炸弹 |
| 深海 | 木镐子 + 横排炸弹 + 九宫格炸弹 |
| 地心火山泥 | 高阶木镐补给 + 双炸弹组合 |

## 10. 视觉反馈与动画清单

### 10.1 环境表现

- 深度加深时背景色和矿层纹理变化
- 灯光、粉尘和矿井支撑结构变化
- 向下推进时有镜头位移感
- 不同深度类型有明确背景区别

### 10.2 挖掘反馈

- 点击砖块时播放敲击反馈
- 破坏成功时砖块碎裂并掉落收益
- 使用炸弹时播放范围爆破反馈
- 木镐点击时，应有单点聚焦高亮或命中闪光
- 横排炸弹和九宫格炸弹释放前后，都应有明确的范围高亮反馈
- 资源掉落后，应有“矿石 / 金币 / 材料”从命中格子飞入顶部累计区的飞入特效

### 10.3 高价值收益反馈

- 矿砖命中时加强音效
- 盲盒矿掉宝箱时强化弹出反馈
- 进入新阶段时做轻量庆祝表现
- 解锁稀有矿石、红砖盲盒矿、宝箱时应有首次出现提示反馈
- 宝箱自动开启时，应有奖励展开并即时叠加反馈

## 11. 交互逻辑

### 11.1 点击砖块挖掘

- 点击可破坏砖块时消耗 `1` 个木镐子并造成 `1` 点伤害
- 点击不可敲砖时只给无效反馈，不消耗木镐子
- 如果该格仍有深红遮挡层，则本次点击只能先作用于遮挡层

### 11.2 可挖掘范围与连通规则

- 木镐只能点击“与当前坑道连通”的格
- 连通判定采用四方向邻接，不采用斜向连通
- 可视范围内的格，不等于当前可直接挖掘的格
- 未与当前坑道连通的格，即使玩家已经看见，也不能被木镐直接点击
- 最底行与其他可视格完全同规则；若其与当前坑道连通，则同样可以直接点击挖掘
- 若格子上仍有深红遮挡层，则本次点击只能作用于遮挡层本身，不能越过遮挡层直接伤害第二层内容
- 深红遮挡层破坏后，格子下方真实内容才转为可继续点击状态
- 生成新行时，必须保证玩家始终至少存在一条继续向下扩张的合法操作链

### 11.3 炸弹交互

- 先点击炸弹图标，再点击目标区域释放
- 释放前显示范围预览
- 若目标不可释放，则给出无效提示
- 横排炸弹的范围预览需覆盖当前目标所在整行
- 九宫格炸弹的范围预览需覆盖目标周围 `3×3` 区域
- 范围预览与实际爆炸命中区域必须完全一致
- 炸弹可以波及未连通但可见的格
- 但炸弹本身的施放落点必须合法
- 合法落点定义为：必须在当前“可连通的坑道上”
- 炸弹释放落点限定为当前可连通坑道中的合法操作格
- 如果某个位置不满足“当前可连通坑道”条件，则即使它在炸弹范围预览上看起来可见，也不能作为炸弹释放落点

### 11.4 结算交互

- 点击 `收队返场` 后先进入确认弹窗
- 确认返场后直接进入结算弹窗
- 工具耗尽后直接进入结算弹窗

### 11.5 继续下挖交互

- 清理当前区域后，画面自动向下补充新砖块
- 达到更深阶段时弹出阶段提示

## 12. 边界与异常处理

- 体力不足时，入口置灰并提示 `体力不足`
- 主背包满时，需按主系统规则处理，不在本模式内单独兜底
- 结算中断线时，按服务端已记录收益补发
- 玩家退出游戏后，未结算收益按主系统规则恢复或结算

## 13. P0 MVP 范围

### 13.1 P0 必须实现

- `1` 点体力 = `1` 局
- 每局固定 `20` 个木镐子
- 每局固定 `1` 个横排炸弹
- 每局固定 `1` 个九宫格炸弹
- 5 类砖块
- 3 类工具
- 深度持续生成
- 退出机制仅包含主动退出和工具耗尽
- 局中补给来源为积分 bar、工具格、宝箱
- 稀有矿石、红砖盲盒矿、宝箱按深度阈值逐步解锁
- 稀有矿石和红砖盲盒矿的出现概率随深度提升
- 积分 bar 门槛随深度提升，奖励随深度类型变丰富
- 深度类型至少区分地表矿区、深地表、浅海、深海、地心火山泥
- 深度加深后砖块地图分布难度提升
- 工具耗尽直接结算
- 收益统一进入主矿场资源体系
- 视图中明确提示当前深度、本局推进、本局收获
- 木镐只能点击与当前坑道连通的格
- 深红遮挡层必须先破坏
- 炸弹可以波及未连通可视格，但炸弹落点必须在可连通坑道上
- 最底行与其他可视格同规则

### 13.2 P0 暂不包含

- 复杂局内商店系统
- 特殊事件层
- 路线分支层
- 多种失败惩罚
- 独立成就体系

## 14. 验证重点

### 14.1 玩家是否理解单局成本

重点验证玩家是否能快速理解：

- 进入一次要消耗 1 点体力
- 每局开局固定拥有 20 个木镐子、1 个横排炸弹、1 个九宫格炸弹

### 14.2 玩家是否理解深挖收益

重点验证玩家是否能感知：

- 越深层越值钱
- 同样 1 个木镐子在深层更有收益价值
- 达到更深层后会逐步解锁稀有矿石、红砖盲盒矿和宝箱

### 14.3 玩家是否理解“可挖掘范围”和“炸弹落点”

重点验证玩家是否能理解：

- 可见格子中，只有与当前坑道连通的格子可以被木镐直接点击
- 木镐只能挖与当前坑道连通的格
- 炸弹虽然能炸到未连通可视格，但炸弹不能随便落在封闭格上
- 深红遮挡层必须先破坏，不能跨层直击下层内容

### 14.4 工具耗尽后的结束反馈是否成立

重点验证玩家是否会在：

- 主动止盈
- 工具耗尽结算

之间形成清晰认知，明确结束时机。

### 14.5 单局时长是否适合移动端

重点验证：

- 固定 20 个木镐子和两种炸弹各 1 个开局是否让节奏足够紧凑
- 在当前固定资源规则下是否仍适合竖屏移动端短时体验

## 15. 一句话总结

无尽矿井是矿场增长游戏中的主动下挖模式：玩家消耗 `1` 点体力进入一局，系统固定发放 `20` 个木镐子、`1` 个横排炸弹和 `1` 个九宫格炸弹作为基础探索资源，在深度推进和收益止盈的选择中，把本局产出稳定回流到主矿场资源体系；其中木镐的可挖掘目标和炸弹的合法落点必须受到“当前可连通坑道”规则约束。
