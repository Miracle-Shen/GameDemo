## 玩法摘要

无尽矿井是矿场增长游戏中的主动操作扩展模式，玩家从主矿区进入单局挑战，通过点击挖掘、工具使用、掉落获取与向下推进形成核心体验；单局结束后，获得内容回流主矿场既有资源体系。

本摘要仅用于建立玩法方向理解，不作为规则、数值、页面或验收标准依据；所有具体定义以用户 PRD 为准。

## 视觉与情绪基调

整体视觉以竖屏移动端矿井场景为核心，强调“中部棋盘优先、矿井环境包裹、深度推进可感知”。画面应呈现岩层和矿层变化、木制支撑结构、灯光、尘土粒子、不同深度类型的环境色与背景差异。挖掘反馈突出砖块碎裂、命中闪光、范围炸弹预览与爆破命中特效；资源掉落以金币、矿石、材料从格子飞入顶部累计区的动线表现。已挖空区域需要有下陷坑道感，未敲通土层需要有上浮厚重感，深度线和侧边深度牌贴近棋盘展示，形成持续向下探索的视觉记忆点。
