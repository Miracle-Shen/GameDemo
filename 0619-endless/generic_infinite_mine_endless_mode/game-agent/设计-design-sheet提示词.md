📋 🇨🇳:
这是一张横向 3:2 画板，展示《无限矿井》无尽模式 4 个核心场景的 UI 缩略图；缩略图外区域是纯绿色 #00FF00 画板背景，缩略图内部是正式游戏 UI 成片截图。

单行排布：4 个严格 9:16 竖屏缩略图在 3:2 画板上单行横向排列，每个缩略图上下左右留清晰的纯绿 #00FF00 间隙。各格尺寸一致、水平等距、垂直居中。宁可格子小一些、绿边多一些，也严禁为填满高度或宽度把缩略图拉高、压宽、贴边。

核心场景清单：
（1）矿区主页入口（MineHomePage / mine_home）——顶部 HUD 包含返回、体力、金币、玉币、设置；中部是主矿区背景与无尽矿井入口卡，入口卡包含入口标题、消耗 1 点体力提示和进入无尽矿井按钮；底部是主矿场导航与背包入口。
（2）无尽矿井玩法页（EndlessMinePage / endless_mine）——顶部 HUD 与主系统资源区延续主页；状态信息区展示当前深度类型、本局推进、本局状态；本局收获累计区展示金币、矿石、材料；中部矿井主场景承载补给积分条、7×6 挖掘棋盘、深度线、深度侧牌、真实下一行矿井区域、右侧工具条；底部安全操作区包含背包入口和收队返场按钮。
（3）本局收获弹窗母版（ModalResult / modal_result）——完整竖屏上下文中展示暗化背景和中央单个本局收获弹窗，弹窗包含标题、本局深度结果、金币/矿石/材料收获列表、返回矿区主页按钮和关闭按钮。
（4）返回确认弹窗母版（ModalConfirmReturn / modal_confirm_return）——完整竖屏上下文中展示暗化背景和中央单个返回确认弹窗，弹窗包含“确定收队返场？”标题、收益立即结算说明、继续下挖按钮、确认返场按钮和关闭按钮。

【跨页统一】
这 4 个场景共同覆盖全部跨页共享元素：返回按钮图标、设置按钮图标、体力图标、金币图标、玉币图标、矿石图标、材料图标、背包图标、关闭按钮、弹窗主按钮底板、顶部 HUD、底部操作/导航底板、弹窗 chrome、通用按钮体系、卡框体系。同一共享元素在所有缩略图中保持相同造型、色相、切面结构、笔触和材质；同一类资源图标、按钮、卡片、弹窗边框和字体气质保持一致。基础页面缩略图只展示各自默认态；弹窗母版缩略图只展示一个中央弹窗。

【质量要求】
每个缩略图都是高完成度 UI 设计稿，像商业上线项目截图；画面干净，主体突出，信息清晰；跨页共享元素统一；分辨率 2K；画布比例 3:2，3072×2048px。

【负面约束】
no annotation, no label, no callout, no redline, no measurement line, no blueprint, no wireframe, no technical notes, no overlay text, no coordinates, no dimensions, no numbering marks, no bounding boxes, no guide lines, no rulers, no arrows, no filenames, no variable names, no English placeholders。不要出现设计说明层、坐标文字、尺寸文字、编号、辅助线、标尺、指示箭头。基础页面（page_default）缩略图内不绘制任何弹窗、遮罩、toast、跑马灯、结算浮层。不要把缩略图拉伸成横向，不要让 UI 填满非 9:16 单元格；每个缩略图必须保持 9:16 竖屏，四周留绿。
