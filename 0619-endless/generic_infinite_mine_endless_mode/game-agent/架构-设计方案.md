### 页面列表

| 页面 | 代码标识 | 路由状态 | 职责 | 主要区域 |
|------|---------|---------|------|---------|
| 矿区主页入口 | `MineHomePage` | `mine_home` | 展示主矿区资源、无尽矿井入口、体力不足态与入场防抖 | TopHud、HomeScene、EndlessEntryCard、BottomDock |
| 无尽矿井玩法页 | `EndlessMinePage` | `endless_mine` | 承载单局挖掘、工具、深度推进、局内收益累计、主动收队 | TopHud、StatusInfo、RunHarvest、MineScene、DigGrid、ToolBar、BottomAction |
| 返回确认弹窗母版 | `ModalConfirmReturn` | `modal_confirm_return` | 点击左上返回时确认是否立即结算返场 | 暗化遮罩、弹窗底板、标题、说明、双按钮 |
| 本局收获弹窗母版 | `ModalResult` | `modal_result` | 展示主动退出、工具耗尽或中断补发后的本局收获汇总 | 暗化遮罩、弹窗底板、深度结果、收益列表、返回按钮 |
| 拦截提示弹窗母版 | `ModalBlocker` | `modal_blocker` | 承载体力不足、工具不可释放、背包满等阻断或提示 | 暗化遮罩、弹窗底板、提示文案、确认按钮 |

## 动画参数

| 动画 | 类型 | 时长 | 缓动 | 说明 |
|------|------|------|------|------|
| 入场按钮点击反馈 | CSS | 120ms | ease-out | 入口点击后给即时按压反馈，入场中禁用重复触发 |
| 砖块命中闪光 | Pixi | 120-180ms | ease-out | 木镐命中可破坏格时播放 |
| 砖块碎裂 | Pixi | 220-320ms | ease-out | 砖块耐久归零时播放碎裂和消失 |
| 石砖矿裂开 | Pixi | 180-240ms | ease-out | 石砖矿第一次命中后进入裂开态 |
| 炸弹范围预览 | Pixi | 循环 600ms | ease-in-out | 选中横排炸弹/九宫格炸弹并指向目标区域时展示 |
| 爆炸命中特效 | Pixi | 300-450ms | ease-out | 释放炸弹后覆盖命中格播放 |
| 资源飞入顶部累计区 | Pixi + DOM 锚点 | 450-650ms | cubic-bezier(0.22,1,0.36,1) | 金币、矿石、材料从格子飞到对应 RunHarvest 槽位 |
| 顶部累计区数字跳动 | CSS | 180-260ms | ease-out | 对应槽位数值更新时触发 |
| 阶段提升提示 | CSS | 1200-1600ms | ease-in-out | 达到新深度类型后短时显示，结束回到推进中 |
| toast 提示 | CSS | 1200-1800ms | ease-in-out | 稀有矿石、红砖盲盒矿、宝箱首次出现等提示 |
| 弹窗出现/关闭 | CSS | 180-240ms | ease-out/ease-in | 全局 ModalLayer 弹窗母版使用 |
